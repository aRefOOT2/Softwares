Console Controll Commands
cccについて
♦作者: tsesuv notsel
♦作成日時: 2025年6月16日
===========================
Console Controll Commands、cccは、TUIアプリケーション用のTUIライブラリです。
作者がライブラリを作りたい、エディタを作りたいと思ったため開発することになりました。
cccには次の関数が用意されています。なお、カーソルの位置などでは、コンソールの最上行の最も左の位置を`0, 0`として扱います。
また、ここには載っていない関数があるかもしれません。その関数は恐らくデバッグやテスト用に組み立てた後消し忘れたか、ここに載せ忘れた関数です。ご了承ください。

・int cls(void)
  画面をクリアする関数です。`printf("\x1b[2J");`を内部で呼び出しています。
・int setmode(int set)
  引数`set`で指定されたモードに変更します。1の場合は非カノニカルモードに、0の場合はカノニカルモードに変更されます。
・int setcclr(int cc)
  引数`cc`で指定された色コードをもとに文字色を設定します。現在のところはANSIエスケープシーケンスで指定できる8つの色に対応しています。256などのほかの色については将来対応するまでは自力でお願いします。
・int setbclr(int cc)
  引数`cc`で指定された色コードをもとに背景色を設定します。`setcclr`関数と同じく現在は8色のみ対応です。
・int setstyle(int sc)
  引数`sc`で指定されたスタイルコードをもとにテキストのスタイルを設定します。
・int resettxt(int mode)
  `setcclr`、`setbclr`で設定された文字色、背景色をリセットします。両方、片方どちらかを指定できます。両方リセットの場合は`setstyle`で設定したスタイルもリセットされます。
・int getrow(void)
  コンソールの行数を獲得します。返り値で行数を返します。エラーの場合は-1を返します。
・int getcol(void)
  コンソールの横幅を獲得します。返り値で横幅を返します。エラーの場合は-1を返します。
・int getposx(void)
  カーソルのx座標を獲得します。返り値で座標を返します。エラーの場合は-1を返します。
・int getposy(void)
  カーソルのy座標を返します。返り値で座標を返します。エラーの場合は-1を返します。
・int acs(int x, int y)
  カーソルをx, yの位置に移動させます。絶対値で移動します。エラーの場合は1を返します。
・int rcs(int dx, int dy)
  カーソルをdx, dyだけ変えた位置に移動させます。相対値で移動します。エラーの場合は1を返します。
・int bprint(const char *text, ...)
  文字列`text`を出力します。printfのラッパー関数です。文字列出力後、カーソルの位置を出力直前の位置に戻します。
・int bprintln(const char *text, ...)
  文字列`text`を出力します。printfのラッパー関数です。文字列出力後、カーソルの位置を出力直前の位置の1行下に戻します。
・int box(int sx, int sy, int ex, int ey)
  ♦現在バグありです。バグ修正が終わるまでは利用しないように注意してください。♦
  sx, syからex, eyまでの位置にボックスを描きます。
・int kbhit(void)
  キーボードが押されたかどうかを検出します。キーの種類は調べません。押されている場合は1を、押されていない場合は0を返します。
・int csloff(int mode)
  カーソルを非表示にするかを設定します。modeが1の場合は非表示に、0の場合は表示に設定します。
